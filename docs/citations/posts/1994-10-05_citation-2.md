Q : (L) Quel est votre nom ?
R : Donarra.
Q : (L) D’où êtes-vous ?
R : Cassiopée.
Q : (L) Pourquoi à chaque fois que nous communiquons avec vous, les Cassiopéens, avons-nous affaire à une personne différente ?
R : Distribution d’énergie.
Q : (L) À qui l’énergie est-elle distribuée ?
R : Le prochain dans l’ordre.
Q : (L) Cela veut-il dire que vous utilisez notre énergie pour la distribuer ?
R : Non. Nous distribuons.
